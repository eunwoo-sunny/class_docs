[ 깃 설치 ]
1. Mac os 홈브류설치 (홈브류사이트 들어가서 설치명령어 복사해서 터미널에서 실행 5~10분정도 소요)
2. git-scm.com들어가서 깃설치 명령어를 터미널에 실행
3. $git이라고 쳤을때 git관련 명령어들 나오면 설치 잘 된것임



[ 로컬에서 버전관리 순서 ]

1. 버전관리할 폴더생성
2. $git init
3. $git remote add origin 깃허브원격저장소URL -> 원격주소 잘못했을때 지우기 $git remote remove origin 깃허브원격주소URL
4. $ls -al로 .git폴더생성확인
5. $git remote 로 원격연결되었는지 확인(결과가 origin 이라고 뜨면됨)
6. $git status 상태확인(붉은색 파일명 나오면 add가 필요함)
7. $git add .  또는 $git add 원하는파일명 -> 스테이지에 올리기
8. $git status 상태확인(초록색으로 파일명나오면 됨)
9. $git commit -m '커밋메세지 작성'
10. $git push origin master 원격저장소로 푸쉬.이때 원격저장소이름은 바뀔수있음. $git branch로 브랜치명 확인하고 하면됨

[ 원격저장소에서 클론해서 버전관리 시작하기 순서 ]

1. $git clone 깃허브원격저장소URL



[ 터미널 명령어 ]
1. 최상위 폴더로 이동
$cd / 

2. 한단계 상위폴더로 이동
$cd ..

3. 원하는 하위폴더로 이동
$cd 폴더명  또는 $cd 폴더명/폴더명/폴더명

4. 화면지우기
$clear

5. 터미널 종료하기
$exit

6. 파일생성 또는 수정
$vim 파일명 -> 영문 i눌러서 입력모드로 전환하여 텍스트 수정후 -> ESC + :wq로 저장하고 나오기

7. 파일내용보기
$cat 파일명

8. 폴더안에 내용보기
$ls 또는 $ls -al(숨김폴더까지 보여줌)

9. 디렉토리 생성
$mkdir 폴더명

10. 폴더전체 삭제
$rm -rf 폴더이름

11. 현재폴더위치보기
$pwd
